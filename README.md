# mchambar
ejemplo de practica de simulacion de un entorno de mineria, simulacion de datos simples, conecta a firebase tras cada click y luego se puede evaluar acciones de correccion desde google collab, simulando jupiter notebook, imlementaciones que despues pueden ser cargadas en un pipeline de Apache Airflow
